# 🌾 Agricultural Post-Harvest Loss (PHL) Analysis Dashboard

An interactive **Power BI** dashboard designed to analyze post-harvest losses across agricultural value chains. The dashboard provides insights into crop spoilage, storage methods, transportation, regional performance, and factors contributing to post-harvest losses.

---

## 📖 Project Overview

Post-harvest loss is a major challenge affecting food security and agricultural productivity. This Power BI dashboard analyzes agricultural data to identify patterns, measure losses, and highlight opportunities to improve storage, transportation, and supply chain efficiency.

---

## 🎯 Objectives

- Analyze post-harvest losses across regions
- Identify crops with the highest spoilage rates
- Evaluate the impact of storage methods
- Assess transportation efficiency
- Compare seasonal trends in crop losses
- Support data-driven agricultural decision-making
